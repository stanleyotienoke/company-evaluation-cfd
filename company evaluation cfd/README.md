## Overview
This project demonstrates the use of **Discounted Cash Flow (DCF) analysis** to estimate the intrinsic value of a company.  
DCF is a fundamental valuation method in finance, where future cash flows are projected and discounted back to present value using a chosen discount rate.

The project applies DCF modelling to real company data (e.g., Safaricom PLC, Equity Bank, or any other firm of interest) and showcases how financial analysts use forecasting and valuation techniques to support investment decisions.
